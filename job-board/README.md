Job Board Application

This project is a simple job board built as part of a technical assignment.

Features:
- View available jobs
- Filter jobs by department and location
- View job details
- Apply for a job using a form
- View submitted applications for the current session

Tech Stack:
- React
- Vite
- React Router

Getting Started:
1. Install dependencies
   npm install

2. Run the project
   npm run dev
